R 4.4.0, to be released tomorrow, comes with experimental native support for 64-bit ARM Windows machines (aarch64, arm64). Rtools44 with native support for the platform has been released at the beginning of March. The effort to add Windows/aarch64 support to R has been announced in August, 2023.
