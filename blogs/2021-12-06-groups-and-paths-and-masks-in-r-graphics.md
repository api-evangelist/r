UPDATE: (2023-05-18) The behaviour of compositing operators was modified in R version 4.3.0 (affecting the “clear” and “source” operators). The examples in this post have been updated so that they produce the same output (just using a different operator). Support for gradient fills, pattern fills, clipping paths and masks was added to the R graphics engine in R version 4.1.0.
